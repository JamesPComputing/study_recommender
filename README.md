# Study Resource Recommender
TM470 Final Project — James Powell (J6310164)

A web-based study resource recommender system built with Flask and Scikit-learn.
Uses TF-IDF vectorisation and cosine similarity to match user topic interests 
against a curated database of Computing and IT study resources.

## Technologies
- Python 3.14
- Flask
- Scikit-learn (TF-IDF, cosine similarity)
- SQLite
- Jinja2
